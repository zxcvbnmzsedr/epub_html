介绍

为了回馈加入 知识星球 的小伙伴，这两年多的时间我还贡献了一份 《Java 面试指北》 ，你可以将其看作是 JavaGuide 开源版的补充完善，两者配合使用。


目前的话，JavaGuide 还在持续更新完善中！大概一周会更新 1~3 篇文章，



我写了数十篇文章手把手教你如何准备面试：



技术面试题篇的内容和 JavaGuide 开源版本互补，不仅仅包括最基本的 Java、常见框架等八股文，还包括系统设计、分布式、高并发等进阶内容。



为了让小伙伴们自测以检查自己的掌握情况，我还推出了 「技术面试题自测」 系列。不过，目前只更新了 Java 部分的自测，正在持续更新中。




除了《Java 面试指北》之外，星球还有 《Java 必读源码系列》 （目前已经整理了 Dubbo 2.6.x 、Netty 4.x、SpringBoot2.1 的源码）、 《从零开始写一个 RPC 框架》 （已更新完） 、 《Kafka 常见面试题/知识点总结》 等多个专属小册。



这些小册的内容几乎都是我利用业余时间写的，少部分内容是我整理的比如《Java 必读源码系列》这个小册。
